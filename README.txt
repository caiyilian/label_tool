# 课堂行为分析工具

## 使用说明

### 启动程序

双击运行 `课堂行为分析工具.exe`，浏览器会自动打开功能选择页面。

### 功能说明

本工具包含两个功能模块：

#### 1. 视频对齐与抽帧

将前后视角视频按时间对齐，逐秒抽取帧图片。

**使用步骤：**
1. 选择前视角视频文件
2. 设置第一帧对应的真实时间（时:分:秒）
3. 选择后视角视频文件
4. 设置第一帧对应的真实时间（时:分:秒）
5. 选择输出目录
6. 点击"开始同步抽帧"

**输出结果：**
- `forward/` 目录：前视角帧图片
- `backward/` 目录：后视角帧图片

#### 2. 坐姿标注

对抽帧后的图片进行逐帧标注。

**使用步骤：**
1. 选择包含 `forward/` 和 `backward/` 子目录的数据文件夹
2. 在前视角画面上画框标注每个学生
3. 填写状态标签、坐姿标签等
4. 可选：标注姿态关键点
5. 按 `C` 键复制上一帧，微调有变化的学生
6. 重复直到所有帧标注完成

**标注内容：**
- 检测框：框住学生身体
- 状态标签：坐-听说 / 坐-读写 / 站立 / 离席
- 坐姿标签：正常 / 不良
- 不良原因：歪头 / 低头 / 俯身 / 侧身
- 姿态点位：8 个上半身关键点（可选）

**常用快捷键：**
- `C`：复制上一帧
- `K`：进入点位标注模式
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做
- `←` / `→`：上一帧 / 下一帧
- `R`：重置视图

### 注意事项

1. 视频文件名不能包含中文
2. ffmpeg.exe 必须与主程序在同一目录
3. 标注数据自动保存在 `annotations/` 目录下
4. 详细说明请参阅《使用文档.md》
