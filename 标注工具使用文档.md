# 坐姿标注工具使用文档

## 1. 工具概述

坐姿标注工具是一个 Web 端应用，用于对课堂监控视频抽帧后的图片进行逐帧标注。标注结果用于训练"多视角学生行为时序识别模型"，支持课堂坐姿检测和课间行为识别。

### 1.1 启动方式

```bash
python annotation_app.py
```

启动后会弹出控制台窗口，浏览器自动打开标注界面。关闭控制台窗口即停止服务。

### 1.2 数据要求

选择的数据文件夹需满足以下条件：

```
数据文件夹/
├── forward/          # 前视角抽帧图片
│   ├── front_000001.jpg
│   ├── front_000002.jpg
│   └── ...
├── backward/         # 后视角抽帧图片
│   ├── back_000001.jpg
│   ├── back_000002.jpg
│   └── ...
└── annotations/      # 标注结果（自动创建）
    ├── front_000001.json
    └── ...
```

- 前后视角图片数量必须一致
- 图片由抽帧工具（`web_app.py`）生成，每秒一帧
- 文件名不能包含中文

## 2. 标注内容与要求

### 2.1 每个学生需要标注的字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 检测框 (bbox) | 画框 | 用矩形框框住学生身体，从头顶到臀部 |
| 学生编号 (student_id) | 自动/选择 | 同一学生在整段视频中编号保持一致 |
| 状态标签 (status) | 单选 | 坐-听说 / 坐-读写 / 站立 / 离席 |
| 坐姿标签 (posture) | 单选 | 正常 / 不良 |
| 不良原因 (bad_reason) | 多选 | 歪头 / 低头 / 俯身 / 侧身（仅坐姿=不良时） |
| 前视角姿态点位 | 画点 | 8 个关键点（可选） |
| 后视角姿态点位 | 画点 | 8 个关键点（可选，需先画后视角框） |
| 备注 (note) | 文本 | 可选，特殊情况说明 |

### 2.2 状态标签定义

| 状态 | 含义 | 典型场景 |
|------|------|----------|
| 坐-听说 | 坐着听老师讲课 | 老师讲授、学生抬头看黑板 |
| 坐-读写 | 坐着读书或写字 | 学生低头看书、写字 |
| 站立 | 站着 | 回答问题、站起来活动 |
| 离席 | 不在座位上 | 离开教室、在教室走动 |

### 2.3 坐姿标签定义

| 坐姿 | 含义 |
|------|------|
| 正常 | 坐姿端正，头部和身体基本保持正直 |
| 不良 | 存在明显的不良坐姿表现 |

### 2.4 不良原因定义

当坐姿标记为"不良"时，需要勾选具体原因：

| 原因 | 含义 | 典型表现 |
|------|------|----------|
| 歪头 | 头部向左或右倾斜 | 头部不正，偏向一侧 |
| 低头 | 头部向下低垂 | 头部低于正常位置，常见于看手机或打瞌睡 |
| 俯身 | 上身向前弯曲 | 上半身趴在桌上或严重前倾 |
| 侧身 | 身体向左或右倾斜 | 身体不正，偏向一侧 |

### 2.5 姿态点位定义

8 个上半身关键点，按标注顺序：

| 序号 | 名称 | 位置 | 标注要点 |
|------|------|------|----------|
| 1 | 头 | 头顶正中 | 发际线附近 |
| 2 | 脖子 | 颈部与躯干连接处 | 颈部根部 |
| 3 | 右肩 | 右侧肩关节 | 肩膀最外侧 |
| 4 | 右肘 | 右侧肘关节 | 手臂弯曲处 |
| 5 | 右手 | 右侧手腕或手掌 | 手腕位置 |
| 6 | 左肩 | 左侧肩关节 | 肩膀最外侧 |
| 7 | 左肘 | 左侧肘关节 | 手臂弯曲处 |
| 8 | 左手 | 左侧手腕或手掌 | 手腕位置 |

骨架连接关系：
```
        头
         │
        脖子
       ╱    ╲
   右肩      左肩
      │        │
   右肘      左肘
      │        │
   右手      左手
```

**注意**：
- 被遮挡的点可以跳过（标记为不可见）
- 前视角看不到的点，可以在后视角标注（需先画后视角框）
- 前后视角的点位独立存储，不会跨视角连线

## 3. 标注流程

### 3.1 整体流程

```
1. 选择数据文件夹
      ↓
2. 加载第一帧图片
      ↓
3. 对当前帧中每个可见学生依次标注：
   ① 画框 → ② 调整编号 → ③ 填写标签 → ④ 标注点位（可选）
      ↓
4. 所有学生标注完毕 → 点击"下一帧"或按 → 键
      ↓
5. 重复步骤 3-4，直到所有帧标注完成
      ↓
6. 标注数据自动保存到 annotations/ 目录
```

### 3.2 单个学生的标注步骤

#### 步骤 1：画框

1. 在前视角画面上，按住鼠标左键拖拽，画出一个矩形框
2. 框的范围：从头顶到臀部下方，尽量贴合身体轮廓
3. 松开鼠标后，系统自动分配编号（S01、S02...），并弹出标签面板

#### 步骤 2：填写标签

标签面板会自动弹出，所有修改即时生效，无需点击确认：

1. **选择状态**：点击"坐-听说"、"坐-读写"、"站立"或"离席"
2. **选择坐姿**：点击"正常"或"不良"
3. **选择不良原因**（仅坐姿=不良时）：勾选"歪头"、"低头"、"俯身"、"侧身"
4. **填写备注**（可选）：输入特殊情况说明

#### 步骤 3：标注姿态点位（可选）

1. 点击标签面板中的"前视角"按钮，或按 `K` 键进入点位标注模式
2. 按顺序点击放置 8 个关键点：头 → 脖子 → 右肩 → 右肘 → 右手 → 左肩 → 左肘 → 左手
3. 被遮挡的点：右键点击跳过
4. 点击框外区域或按 `Esc` 退出点位标注模式
5. 如需在后视角标注：
   - 先在后视角画一个框（选中学生后，在后视角拖拽画框）
   - 点击标签面板中的"后视角"按钮，或按 `Shift+K` 键
   - 在后视角框内标注点位

#### 步骤 4：处理下一个学生

1. 在前视角空白区域点击，取消当前选中
2. 重复步骤 1-3，标注下一个学生

### 3.3 高效标注技巧

#### 复制上一帧（核心效率功能）

课堂场景中，学生位置和姿态变化缓慢。大部分帧可以直接复制上一帧的标注，然后微调有变化的学生。

1. 按 `C` 键，或点击"复制上帧"按钮
2. 上一帧的所有标注（框 + 标签 + 点位）会复制到当前帧
3. 检查是否有学生位置或状态变化，有变化的进行微调

#### 快速切换状态

- 点击标签面板中的状态按钮，即时生效
- 无需点击确认

#### 批量处理流程

1. 标注第 1 帧（最慢，需要画所有框）
2. 按 → 切换到第 2 帧
3. 按 `C` 复制上一帧
4. 检查并微调有变化的学生
5. 重复步骤 2-4

### 4.3 点位标注模式

进入点位标注模式后，底部会显示提示条：

```
[前视角] 点位标注: 头 (1/8) — 左键放置 | 右键跳过 | Esc退出
```

或：

```
[后视角] 点位标注: 脖子 (2/8) — 左键放置 | 右键跳过 | Esc退出
```

### 4.4 后视角框

- 选中前视角框时，后视角会显示对应的后视角框和点位
- 取消选中时，后视角框和点位隐藏
- 后视角框支持移动和缩放（8 个手柄）
- 右键后视角框可删除

## 5. 快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| 鼠标拖拽 | 画框（前视角）/ 画后视角框（需先选中前视角框） |
| 双击框 | 打开标签面板 |
| K | 进入前视角点位标注模式 |
| Shift+K | 进入后视角点位标注模式 |
| C | 复制上一帧 |
| Del / Backspace | 删除选中标注 |
| Ctrl+Z | 撤销（非标注模式：整体操作；标注模式：逐点撤销） |
| Ctrl+Y | 重做（非标注模式：整体操作；标注模式：逐点重做） |
| R | 重置视图（恢复图片初始位置和大小） |
| ← / → | 上一帧 / 下一帧 |
| Esc | 关闭面板 / 退出点位模式 / 取消选中 |
| 点击框外 | 退出点位标注模式 |
| 滚轮 | 缩放图片 |
| 中键拖拽 / Alt+拖拽 | 平移图片 |

## 6. 数据存储

### 6.1 标注文件

每帧的标注保存为一个 JSON 文件，存储在 `annotations/` 目录下：

```
annotations/
├── front_000001.json
├── front_000002.json
└── ...
```

- 切换帧时自动保存当前帧

- 空标注（无学生）不保存 JSON 文件

## 7. 注意事项

### 7.1 标注规范

1. **框要贴合**：画框时尽量贴合学生身体轮廓，不要太大或太小
2. **编号一致**：同一学生在整段视频中使用相同编号，不要重复或遗漏
3. **状态准确**：根据学生当前动作准确选择状态标签
4. **不良原因要选**：标记为"不良"时，必须勾选至少一个不良原因
5. **点位可选**：姿态点位是可选的，根据需要标注

### 7.2 常见问题

**Q: 标注过程中 annotations 文件夹被删除了怎么办？**
A: 工具会自动重建文件夹，继续保存当前帧。但之前已标注的帧数据会丢失，帧列表会自动同步状态。

**Q: 后视角的框和点位怎么不显示？**
A: 后视角框和点位只在选中前视角对应框时才显示。点击前视角框即可显示。

**Q: 如何修改已标注学生的编号？**
A: 双击框打开标签面板，在"学生编号"下拉框中选择新编号。如果新编号已被其他学生使用，会自动交换两个学生的编号。

**Q: 点位标注时标错了怎么办？**
A: 按 `Ctrl+Z` 撤销上一个点位，可连续多次撤销。按 `Ctrl+Y` 重做。

**Q: 如何只标注部分点位？**
A: 可以只标注可见的点，被遮挡的点右键跳过。也可以只在前视角或后视角标注，不要求两个视角都标。

### 7.3 效率建议

1. **先标第 1 帧**：第 1 帧需要画所有框，是最慢的一帧
2. **大量使用复制**：第 2 帧开始，按 `C` 复制上一帧，只调整有变化的学生
3. **善用快捷键**：`K` 标点位、`Del` 删除、`Ctrl+Z` 撤销
4. **后视角辅助**：前视角遮挡的学生，参考后视角来判断状态和坐姿
5. **点位按需标注**：不是每个学生都需要标点位，根据需要选择性标注
